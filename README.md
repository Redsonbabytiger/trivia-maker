# trivia-maker
Test your JavaScript knowledge! Answer JavaScript questions from a difficulty of your choosing. You can also create custom questions or choose from database questions.
The Database questions are publicly submitted questions by people using the custom questions page and submitting them.

Upcoming Features:
- Selecting database questions by category.
- User Account Creation
- Points earned from playing trivia become stored in your account
- Have your username displayed in every trivia set you create
